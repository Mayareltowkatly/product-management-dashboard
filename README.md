# product-management-dashboard

## Overview
- The Product Management Dashboard is a responsive CRUD web application designed to manage products efficiently through a clean and user-friendly interface.
- The application allows users to create, view, update, search, and delete products while storing all data locally using the browser's Local Storage. It also supports image uploads and real-time form validation to ensure accurate user input.

## Features
- Add new products with image upload.
- Edit existing product information.
- Delete individual products.
- Delete all products with a single click.
- Real-time product search by name.
- Store product data using Local Storage.
- Upload and preview product images.
- Form validation using Regular Expressions (Regex).
- Responsive design for desktop and mobile devices.

## Product Information
Each product includes:
- Product Image
- Product Name
- Product Price
- Product Count
- Product Type
- Product Description

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Local Storage API
- Font Awesome
